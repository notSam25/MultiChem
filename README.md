## MultiChem

> A multipurpose chemistry tool, written in c++

## About

    This tool can be used to:
    * Balance chemical equations
    * Mole Conversions
    * Percent Abundance
    * Determine a chemical equations physical states.

## Building

> This program can be built using cmake 3.16+
    ```bash
    mkdir build
    cd build
    cmake ..
    make
    ```
